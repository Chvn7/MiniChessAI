# Gardner Chess (5x5)

Gardner Chess is a 5 by 5 board chess, containing all kinds of chess pieces on the 1st row and pawn on the 2nd row. The game has roughly 9 x 10^18 legal positions, comparable in complexity to Checkers.

> **Note:** This game is officially a **weakly solved game**, meaning perfect play results in a draw. This AI implementation focuses on a challenging experience not the solved approach (AI look 3 moves ahead0

## Gameplay Rules:

* **Castling:** No
* **Pawn Mechanics:**
    * **Promotion:** Yes
    * **Double-Step:** No
    * **En Passant:** No
